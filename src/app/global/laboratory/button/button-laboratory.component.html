Button

Types:

Defines the appereance of the button using the type input with options: 'primary' | 'secondary' | 'success' | 'error'. Default value: 'primary'.

<hec-button type="secondary">Click Me!</hec-button>
Primary
Secondary
Success
Error
Disabled

Variations:

Set up the the variant input with options: 'contained' | 'outline' | 'text'. Default value: 'contained'.

<hec-button type="primary" variant="outline">Click Me!</hec-button>
Contained
Outline
Text

Size:

Set up the the size input with options: 'small' | 'medium' | 'large'. Default value: 'medium'.

<hec-button size="large">Click Me!</hec-button>
Small
Medium
Large

Events work exactly as Angular Material button.
