Radio Button Group

Includes Form Control Validations

Sample:

Send an array of options using the options input, specify form control and type: 'primary' | 'secondary' | 'success' | 'error'. Default value: 'primary'.

<radio-button-group type="primary" [formControl]="formGroupSimple.controls['simpleControl']" [options]="seasons"></hec-button>
Selected: {{ formGroup.controls["seasonControl"].value }}
